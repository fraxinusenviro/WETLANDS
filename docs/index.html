Using WETLANDS NTB

  • Use Launch New Form to start, or Drafts Library to resume prior work.
  • The form autosaves while you type. Returning home prompts to save a resumable snapshot.
  • In soil horizons, enter Munsell codes (e.g., 10YR 4/3) for Matrix/Redox fields.
  • Known Munsell codes show as Code (Description) and PDF exports include these descriptions.
  • Submit when complete, then manage exports in Submitted Forms.
